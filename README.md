<img src="./Programming Hero Assignment 01.png">

## Overview
This is a responsive landing page for **Personal Knowledge Vault**, designed to showcase features of a note-taking and knowledge management platform. The design focuses on clarity, structured thinking, and promoting early access to the product.

The project uses **HTML**, **CSS**, and **Font Awesome icons**, along with **Google Fonts**.

## Features
- Fully responsive landing page
- Header navigation with logo and menu
- Banner section highlighting early access
- About section with feature cards
- Solution section showcasing platform benefits
- Call-to-action sections encouraging users to join
- Footer with social media links
